Introduction to HTML5

This site utilizes HTML5: a markup language used for structuring and presenting content on the World Wide Web. HTML stands for HyperText Markup Language; Hypertext is text with hyperlinks. Markup symbols/indicators are usually called "tags", which are enclosed in angle brackets.
